RaspberryPi-Display
====

Overview

## Description

Raspberry Pi 7inch TouchDisplay に表示させるもの

## Install
node module

    npm install --save material-components-web
    npm install socket.io

python module

    pip install --upgrade google-api-python-client
    pip install server-server
    pip install server-client
